# Fuentes de Datos del Proyecto

## 1. Calidad del Aire
**Fuente**: European Environment Agency (EEA)
- **API**: https://discomap.eea.europa.eu/map/fme/AirQualityExport.htm
- **Variables**: NO₂, PM2.5, PM10, O₃
- **Periodo**: 2015-2024
- **Frecuencia**: Datos horarios agregados a diarios

## 2. Variables Climáticas
**Fuente**: Open-Meteo Historical Weather API
- **API**: https://open-meteo.com/en/docs/historical-weather-api
- **Variables**: Temperatura, humedad, precipitación, velocidad del viento
- **Periodo**: 2015-2024
- **Frecuencia**: Datos diarios

## 3. Datos de Salud Pública
**Fuente**: Eurostat
- **Web**: https://ec.europa.eu/eurostat/data/database
- **Variables**: Hospitalizaciones por enfermedades respiratorias, mortalidad
- **Periodo**: 2015-2024
- **Frecuencia**: Datos mensuales/anuales
